Topology Optimization for Fluid Flows using AI

Optimization · PyTorch · Fluid Dynamics

I began this project last year, aiming to develop an algorithm that generates optimal topologies under specified constraints in fluid flows (for example, aircraft geometry or propeller design). This problem is significantly harder to solve than traditional structural topology optimization because fluid simulations are not differentiable, unlike FEM simulations. My solution was to train a neural network to predict the results of a fluid simulation for a given topology and then optimize using topological gradients. This involved constructing a large dataset of fluid simulations for various shapes using C++, which took several days to complete on my computer. I trained a convolutional network (CNN) to predict fluid forces using Python and Tensorflow, and later converted it to PyTorch.
